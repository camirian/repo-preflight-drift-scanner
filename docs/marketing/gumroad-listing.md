## Product title

AI Agent Repo Preflight Kit

## Subtitle

A local preflight scanner for AI-assisted repositories.

## Price target

$9 introductory price.

## Product description

AI coding agents can generate a lot of code quickly. The failure mode is not always a bug. Sometimes the failure is release drift: unchecked gates, risky public claims, generated artifacts, stale reports, missing process files, or public-export hygiene issues.

AI Agent Repo Preflight Kit is a deterministic local CLI and GitHub Action that scans for those obvious release blockers before you merge, demo, publish, or ship.

## You get

- Python CLI scanner
- GitHub Action wrapper
- sample repository
- Markdown, JSON, HTML, and SARIF report outputs
- strict, docs, and public-export profiles
- privacy-first reporting options
- buyer quickstart
- local and GitHub Action setup guides
- sample report walkthrough

## Use it when

- you used an AI coding agent heavily
- you are about to publish a public repo
- you are packaging a template, tool, or Action
- you need a repeatable release preflight before a demo

## Boundary

This is a release-discipline scanner, not a security scanner. It does not guarantee that code is secure, compliant, correct, or production-ready.

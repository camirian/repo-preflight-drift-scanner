## Product title

AI Agent Repo Preflight Kit

## Short description

A deterministic local CLI and GitHub Action for checking obvious release blockers in AI-assisted repositories before you publish.

## Price target

$9 introductory price.

Raise to $19 if the package includes polished screenshots, sample reports, and a stronger walkthrough.

## Description

AI coding agents make it easy to generate code quickly. They also make it easy to publish a repository that still carries release drift: unchecked gates, risky public claims, generated artifacts, stale reports, missing process files, or private-publication hygiene problems.

AI Agent Repo Preflight Kit gives you a deterministic local preflight scanner and GitHub Action for catching those problems before you merge, demo, publish, or ship.

## Included

- local Python CLI
- GitHub Action metadata
- sample repository
- Markdown, JSON, HTML, and SARIF reports
- strict/docs/public-export profiles
- privacy controls for shareable reports
- buyer quickstart
- local CLI setup guide
- GitHub Action setup guide
- limitations and sample report walkthrough

## Best for

- developers using Codex, Claude Code, Cursor, Gemini CLI, or other AI coding agents
- indie hackers publishing public repositories
- founders preparing demo repositories
- engineers packaging GitHub Actions, templates, or downloadable code assets

## Important limitation

This is not a security scanner, compliance scanner, vulnerability scanner, or guarantee of production readiness. It is a deterministic release-discipline preflight.

## Delivery

Downloadable ZIP. No account, SaaS backend, or custom service required.
